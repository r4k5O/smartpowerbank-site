# smartpowerbank
**smartpowerbank** is the worlds first powerbank that not only charges, but also manages your devices! It can effectively save electricity by managing all devices! But that's not everything: Small solar panels and wind turbines help us reduce our ecological footprint and can help you charge the powerbank. smartpowerbank is an DIY project and will not get sold from us.
